# Portfolio

Portfolio website showcasing education, experience, technical skills, projects, certifications, and achievements.

## What this contains

- `index.html` — the main page using the Tailwind Play CDN for styling
- `assets/` — SVG logo and cute illustration

## Run locally

Easiest: open `index.html` in a browser.

Use the included Vite + Tailwind setup (recommended for development):

PowerShell (Windows):

```powershell
cd c:\Users\AditiVanikar\web3\cute-website
npm install
npm run dev
```

Build for production:

```powershell
npm run build
npm run preview
```

## Notes

- The project includes a Vite-powered portfolio website. `src/` contains `main.js` and `styles.css`.
- You can customize the sections in `index.html` to update the resume content and project details.

Enjoy building your portfolio! ✨
